# Regex Search in [[Obsidian]] 
```/d = number
/d/d = number with 2 digits
/d/d% = percentage number with 2 digits 
/d/d%\ Dabi = percentage number with 2 digits mentioned by Dabi
or 
Section:(/d/d%\ Dabi)
```
